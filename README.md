DOS 打字练习软件 TT 的 Node.js 命令行简化版

## Usage
```shell
npm install yzhantt -g
yzhantt
```
`ESC`: Back  
`Enter`: OK  
`↑`: Up  
`↓`: Down  
`Any Letter Key`: Match word from left to right
